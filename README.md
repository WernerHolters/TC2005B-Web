# TC2005B - Construcción de Software y Toma de Decisiones (WEB)

Repositorio con los códigos, proyectos y entregables del curso TC2005B Construcción de Software y Toma de Decisiones (WEB).

Incluye implementaciones de aplicaciones web con JavaScript vanilla, sistemas de autenticación, consumo de APIs REST, gestión de tareas con backend y bases de datos, y aplicaciones interactivas con persistencia local.

## Contenido

### Proyectos

- **app-tareas**: Aplicación de gestión de tareas (Vanilla JS + Express + Postgres)
- **garden-grid**: Simulador de jardín interactivo
- **remember**: Sistema de login y navegación
- **valorant-api**: Visualizador de agentes de Valorant

## Descripción de Proyectos

### app-tareas

Aplicación full-stack para gestionar tareas con estados y asignación de usuarios.

**Tecnologías:** Vanilla JavaScript, Express.js, PostgreSQL

**Características:**
- CRUD completo de tareas
- Gestión de usuarios
- Sistema de estados (sin iniciar, en curso, terminada)
- Búsqueda y filtrado
- Paginación

---

### garden-grid

Simulador interactivo de jardín en formato grid 8x8.

**Tecnologías:** HTML5, CSS3, JavaScript (Vanilla)

**Características:**
- Plantar diferentes tipos de plantas
- Sistema de riego y maduración
- Cosecha de plantas maduras
- Persistencia con localStorage
- Panel lateral con filtros y búsqueda

---

### remember

Sistema de autenticación y navegación web.

**Tecnologías:** HTML5, CSS3, JavaScript (Vanilla)

**Características:**
- Sistema de login con validación
- Navegación entre páginas
- Footer y menú dinámicos
- Diseño responsive
- Formularios de contacto y registro
